# 2SW-2LED バージョン

DigiKeyboardに使用するスイッチを変更し、表面実装のLEDを追加したバージョンです。  
Boothにて依頼を受け作成しました

# 使用方法

ファームウェアをArduino IDEで開き、[Ctrl] [Shift] [N]を同時押しして[Driver.ino]ファイルを作成し、[Driver.ino](./Driver.ino)の中身をコピペしてください
